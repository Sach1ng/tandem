# Drop your assets here

This folder is served by Remotion via `staticFile()`.

- **Recordings:** put your screen captures here (e.g. `slack-demo.mp4`,
  `desktop-demo.mp4`, `browser-demo.mp4`) and set the filenames in
  `../src/recordings.ts`. H.264 `.mp4` is the most reliable.
- **Music:** put a track here (e.g. `music.mp3`) and set it in
  `../src/recordings.ts`.

Until you add files, each demo scene shows a labeled placeholder slot so the
layout and timing are already final.
